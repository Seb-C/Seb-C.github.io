Publication d'une librairie JS : FileLoader.js

Je viens de publier sur GitHub une petite librairie javascript. Je ne vais pas réécrire toute la description que j'y ait mis, donc je vous invite à aller voir :

https://github.com/Seb-C/FileLoader.js

Basiquement, elle vous permet de mettre tous les fichiers nécessaires à votre application web (scripts, images, binaires, ...) dans une archive tar et de tout charger par une unique transaction HTTP. Ça permet dans mon cas d'éviter au client de télécharger les 132 fichiers nécessaires à l'exécution du jeu. Le tout est mis dans un unique fichier tar qui est parsé en JavaScript côté client.
